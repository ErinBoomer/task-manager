# Task Manager Web App

## Description
This is a simple task manager web application built using HTML, CSS, and JavaScript. It allows users to add, edit, delete, and mark tasks as complete. Tasks are stored using LocalStorage for persistence.

##  Features
- Add tasks with date and time
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Persistent storage using LocalStorage

##  Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

##  How to Run
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Start managing tasks


##  Project Structure
- index.html
- style.css
- script.js
